## Installation
--------------

### Prerequisites

*   Java 11 (or later)
*   Maven 3.6.0 (or later)
*   MySQL 8.0 (or later)

### Installation Steps

1.  Clone the repository: `git clone https://github.com/your-username/core-engine.git`
2.  Navigate to the project directory: `cd core-engine`
3.  Run the build command: `mvn clean install`
4.  Create a MySQL database: `mysql -u your-username -p your-database`
5.  Configure the database connection properties in the `application.properties` file
6.  Run the application: `mvn spring-boot:run`

### Running the Tests

1.  Run the test command: `mvn clean test`
